# Fonts

This folder is reserved for bundled open-source Chinese fonts.

The first implementation uses carefully chosen system fallback stacks so the tool works immediately. To bundle fonts later, place licensed `.woff2` files here and add matching `@font-face` rules in `src/styles/app.css`.

Suggested open-source font families to evaluate before bundling:

- Noto Serif SC
- LXGW WenKai
- Zhi Mang Xing
- Ma Shan Zheng
