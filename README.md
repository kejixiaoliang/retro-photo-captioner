# Retro Photo Captioner

A local-first web tool for creating retro Chinese commemorative photos. Upload a photo, add a deep red top or bottom banner, type multi-line Chinese caption text, apply vintage filters, and export PNG or JPEG in the browser.

## Features

- Pure frontend image processing with Canvas.
- Top or bottom red banner without cropping the original photo.
- Editable multi-line Chinese caption text.
- Retro font presets with system fallbacks.
- Multiple vintage filter presets with adjustable strength, grain, and vignette.
- PNG and JPEG export.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
